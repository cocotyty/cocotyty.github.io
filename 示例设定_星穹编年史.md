## 贰 · 世界档案

### 纪元考

**星历 1473 年，大探索时代·第七纪元。**

八百年前，第一座天然灵能水晶矿脉在边缘星系「碎星带」被发现。它像一颗石子投入死水——原本信奉纯粹科技的星海帝国，突然被注入了名为"修真"的变数。灵能水晶释放的「虚场辐射」能让人类肉身感知到宇宙中流淌的「星灵之气」，从此武道崩溃，修真崛起。

帝国科学院将其命名为 **「源能溢出事件」**。

如今的世界，是蒸汽朋克管道与飞剑并存的奇景：巨型星际都市的外壳上，符箓刻印与电路板交错闪烁；太空港的泊位上，灵能帆船与反物质引擎战舰并肩停靠；人们用全息手环聊天，用神识传音谈情说爱。

### 疆域

帝国疆域横跨 **十七条旋臂**，下辖 **三千二百余颗殖民星**，划分为：

- **内环九星** —— 帝国核心，灵气浓郁如浆，高门世家林立
- **中环百域** —— 商业与军事重镇，新旧势力犬牙交错
- **外环星带** —— 蛮荒边境，灵能矿脉与未知遗迹层出不穷，也是故事真正开始的地方

### 大事件

| 时间 | 事件 |
|:---|:---|
| 星历 0 年 | 帝国建国，纯科技文明 |
| 星历 673 年 | 碎星带灵能水晶首次被发现，史称「源能觉醒」 |
| 星历 700 年 | 第一位以修真法门突破金丹的修士出现 |
| 星历 1021 年 | 灵修会从帝国科学院独立，修真成为独立力量体系 |
| 星历 1300 年 | 「寂静边境」战役——帝国与不明外星种族的首次接触，以惨胜告终 |
| 星历 1450 年 | 黯星侯叛乱，外环星带陷入割据 |
| **星历 1473 年（现在）** | 一艘逃逸舱在碎星带边缘漂流，舱内静卧着一个失忆的年轻人 |

### 核心冲突：三股洪流

**科学与修真** 的矛盾只是表象。更深层的冲突在于三种力量对"宇宙真相"的争夺：

1. **帝国正统派** —— 认为灵能只是另一种可量化的自然力，试图将修真纳入科学院管理
2. **灵修会** —— 坚信修真超越物质，追求灵能飞升，视科技为外道
3. **黯星教** —— 新兴的异端教派，宣称在宇宙的尽头发现了"另一侧"的存在，认为现在的世界只是某位大能的梦境

三条道路都将目光投向了**碎星带深处**——那里有尚未被探索的星域，有远古文明的遗迹，有让黯星侯不惜叛变也要得到的秘密。

### 黯星侯 · 凌破虚

> *「你们都以为我疯了。但等我打开那扇门，你们就会跪下叫我——'先知'。」*

#### 档案

| **境界** | 金丹后期 · 兼修殖装强化 |

#### 外貌与气质

身披玄黑大氅，左半脸覆盖着银色灵能殖装——那是他在「寂静边境」战役中负伤后被改造的。殖装上的蓝色光纹随呼吸明灭，如同第二道脉搏。他的独目是深琥珀色，看人时像在透过你的皮肉审视你的灵魂。

他说话极慢，每个字都像经过精密称量。从不发怒，因为发怒意味着失控。

#### 生平历程

| 时期 | 事件 |
|:---|:---|
| 星历 1313 年 | 出生于外环星带的矿工家庭，天资卓绝 |
| 星历 1330 年 | 通过帝国军校选拔，以全科甲等毕业 |
| 星历 1345 年 | 晋升为帝国最年轻的星舰舰长 |
| 星历 1370 年 | 参与「寂静边境」战役，负重伤，接受灵能殖装改造 |
| 星历 1385 年 | 封侯，总督第七星域，开始秘密探索碎星带深处 |
| 星历 1420 年 | 在碎星带发现「镜面遗迹」——一座不属于任何已知文明的建筑 |
| 星历 1440 年 | 声称在遗迹中接触到了"宇宙的真实"，开始秘密组建黯星教 |
| 星历 1450 年 | 叛变，率部众占据外环星带，自称黯星侯 |
| 星历 1473 年（现在） | 计划开启「镜面之门」，迎接"另一侧"的存在降临 |

#### 真实动机

**他并非疯子，而是看见了太多真相的人。**

在「镜面遗迹」中，黯星侯接触到了这个宇宙的底层信息流。他意识到：

> *「我们所认知的'真实'，不过是某个更高维存在的一场实验。灵能、修真、科技——都是实验变量。而我们是培养皿里的细胞。」*

他的计划不是毁灭世界，而是**打破培养皿**。他相信在"另一侧"，人类能以更真实的形态存在。为此，他不惜牺牲整个星系。

#### 邪恶计划 · 五章时间线

| 章 | 计划阶段 | 行动内容 |
|:---|:---|:---|
| **第1章** | 序曲：追猎钥匙 | 追杀携带家传物的主角——那件物品实际上是「镜面遗迹」的共鸣密钥。黯星侯派出座下「三獠牙」追捕，同时在碎星带布置大型灵能共振阵 |
| **第2章** | 升腾：血祭引路 | 在七个殖民星同步发动献祭仪式，以百万生灵的灵魂能量冲击星界壁垒，为"门"的开启积蓄力量。主角的家乡星球被选为第一个祭品 |
| **第3章** | 开门：镜面倒映 | 「镜面之门」在外环星带显形。门另一侧涌出的"虚能辐射"开始扭曲现实物理法则——在这片区域，修真者的力量暴增，科技武器全部失效 |
| **第4章** | 降临：旧日归来 | 门中开始涌出不属于这个维度的存在——「虚境之民」。它们不是来入侵的，而是来"回收实验数据"的。黯星侯发现自己也不过是棋子 |
| **第5章** | 终局：超越之价 | 黯星侯面临选择：继续开门，让虚境之民"格式化"这个宇宙；或者与主角联手封印镜面之门，但代价是永远失去接触"真实"的可能 |

| **核心诉求** | 维持帝国统一，压制灵修会，歼灭黯星教 |

帝国是腐朽而庞大的巨人。表面上依旧威严，但内部派系倾轧严重——科学院与军部的矛盾、世家与寒门的对立、传统派与改革派的撕扯，都在蚕食它的根基。

> 帝国不会帮主角。但主角的存在，可以被帝国当作一把刀。

---

### ② 灵修会 ——「肉体凡胎，终将飞升」

| 项目 | 内容 |
|:---|:---|
| **实力评级** | ★★★★☆（高端战力惊人，但缺乏舰队） |
| **立场** | 中立偏神秘 · 避世修行 |
| **领袖** | 大祭酒 · 云中君（化神初期，已知宇宙最强个体之一） |

灵修会表面上是一个松散的修行者联盟，实则内部有严密的等级制度。他们掌握着帝国都无法完全解读的远古修真典籍，以及灵能水晶的提纯秘法。

**秘密**：灵修会的高层知道"穿越者"的存在，因为第一位修真者——传说中的「开脉真人」——就是一个穿越者。他们一直在寻找来自"另一侧"的灵魂。

> 他们是最疯狂的敌人，但也是最了解真相的人。

### 场景一 · 锈铁棺废弃空间站

> *「它像一具被遗弃在星空中的铁棺。空气是冷的，灯光是断的，但某种东西——某种不该存在的东西——仍在它的血管里跳动。」*

| 项目 | 内容 |
|:---|:---|
| **名称** | 锈铁棺（Rust Coffin） |
| **类型** | 废弃空间站 · 微型副本 |
| **冲突模式** | 资源争夺 + 环境解谜 + 轻度战斗 |
| **氛围** | 冰冷、寂静、压抑，偶尔有金属热胀冷缩的呻吟声在通道中回荡 |
| **建议等级** | 凡体期（主角初始可通关） |

#### 本地故事梗概

七年前，这里是「第七星域矿业公司」的前哨站，负责转运灵能水晶原矿。某一天，所有通讯突然中断。帝国派出的搜救队发现站内空无一人——生活区还摆着吃到一半的饭，控制台的屏幕上跳着未发送的求救信，但所有人像泡沫一样消失了。

官方结论是"黯星教袭击"，但了解内情的人知道不是。

真正的原因是：空间站的钻井意外打通了一个「虚境空洞」——一小片与现实交叠的异次元空间。站内的人不是消失了，而是被"卡"在了两个维度的缝隙里。

他们还在那里。只是你看不见他们。他们也看不见你——但能感觉到你。

#### 探索要素

| 类型 | 数量 | 说明 |
|:---|:---|:---|
| 解决方式种类 | 3 | ① 关闭虚境空洞（需灵能亲和天赋）② 重置空间站能源系统并逃离 ③ 沟通被困者，达成某种交易 |
| 隐藏真相碎片 | 2 | ① 失控钻井日志（提及"镜面遗迹"坐标）② 黯星侯亲笔信（七年前他来过这里，发现了虚境现象——这是"镜面之门"的雏形） |
| 入口线索 | 主角的逃逸舱被空间站的自动回收系统捕获，从废物处理通道进入 |

#### 关键互动点

- **食堂的广播** —— 反复播放着一段星语广播："请到 C 区集合，有重要通知。"循环了七年。按下"回复"键，可能会收到回应
- **虚空裂隙** —— 空间站中央区域有一个肉眼不可见的维度裂缝。用灵能感知可以看到那里"挂着"一群人形剪影，他们在无声地呼喊
- **失控钻机** —— 仍在运转的巨型钻机，钻头深深扎入虚空。如果主角能关闭它，空间站会开始解体，但也会触发最终的真相信息

### 场景二 · 碎星镇

> *「灵能水晶在这里像沙子一样不值钱。值钱的是情报、弹药、和一条通往下个星域的活路。」*

| 项目 | 内容 |
|:---|:---|

| **类型** | 枢纽型定居点 · 安全区 |
| **冲突模式** | 社会博弈 + 情报收集 + 支线任务中心 |
| **氛围** | 嘈杂、混乱、霓虹与灯笼共舞。矿工、佣兵、走私犯、修士、探宝者挤在同样狭窄的街道上 |
| **建议等级** | 任意 |

#### 本地故事梗概

碎星镇诞生于五十年前的一次"灵能水晶潮"。最初只是矿工们的临时营地，后来因为地处三条航道的交汇处，逐渐发展成外环星带最大的非法贸易集散地。

这里没有法律。商盟的规矩就是规矩——不能杀人越货，至少在镇内不行。出了镇界，生死自负。

碎星镇是主角的第一个"落脚点"——这里有修理厂（铁老板）、有情报贩子（商盟）、有潜在的队友（零号可能在这里执行帝国任务），也有追兵（黯星教的三獠牙已经抵达）。

#### 探索要素

| 类型 | 数量 | 说明 |
|:---|:---|:---|
| 解决方式种类 | 5+（非线性的沙盒空间） | ① 结交势力 ② 赚取路费 ③ 打探情报 ④ 购买装备 ⑤ 触发同伴支线 |
| 隐藏真相碎片 | 3 | ① 商盟密档（关于主角家传物的鉴定报告）② 灵修会联络点的暗号 ③ 帝国通缉令的"附加条款"（活捉主角的赏金是死的十倍） |
| 入口线索 | 从锈铁棺逃出后，被商盟巡逻船救起，带到最近的定居点——碎星镇 |

#### 关键互动点

| 地点 | 描述 | 可触发事件 |

| **铁炉堡机械坊** | 吴三斤的店，从殖装维修到武器改装什么都做 | 修复家传物 → 解锁共鸣密钥用途；接取佣兵任务赚钱 |

*以上设定基于「骰子系统」的完整掷骰结果生成。所有元素均为随机组合的结果，但在编织过程中，一些有趣的巧合浮现了出来：*

- *「意识转移」的死亡规则与零号的身世完美契合——她本身就是这条规则的产物*
- *「多时间线」让玄尘子"认识前世的你"这个设定变得合理，也为后续的时间线操作玩法埋下伏笔*
- *「灵能亲和 + 记忆封印」既解释了主角为什么在开场很弱（封印中），又确保了成长空间（解封后潜力巨大）*
